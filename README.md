# Dashboard Design System

Design System oficial do Dashboard da Takeat.

## Objetivo

Este repositório contém:

- Componentes da UI Kit
- Tokens de design
- Tipografia
- Espaçamentos
- Radius
- Temas Light e Dark

## Estrutura

/components
/tokens

## Regras

- Utilizar sempre a cor Primary definida em colors.json
- Utilizar apenas componentes descritos em uikit-ai.json
- Respeitar os espaçamentos definidos em spacing.json
- Respeitar os temas light e dark
